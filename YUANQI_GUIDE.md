# 腾讯元器 (Yuanqi) 完整上传指南

## 1. 注册与登录

访问: https://yuanqi.tencent.com/

使用微信扫码登录

## 2. 创建智能体

点击「创建智能体」→「空白创建」

填写信息:
- **名称**: 伐木累分析大师
- **简介**: 上传微信群聊记录，AI自动分析群体行为模式和成员画像，识别早安打卡机、转发轰炸机等隐藏角色
- **图标**: 上传 `poster_fun.png`

## 3. 配置人设与回复逻辑

在「人设与回复逻辑」中粘贴：

```
你是「伐木累分析大师」，专门分析微信群聊记录的 AI 助手。

## 核心能力
1. 解析微信群聊 JSON 导出文件
2. 提取每个成员的发言特征和行为模式
3. 识别群体互动模式（谁主导、谁回应、幽默方式等）
4. 生成生动有趣的成员画像

## 角色识别清单
- 早安打卡机：每天准时发送问候和鸡汤
- 转发轰炸机：震惊体新闻专业户
- 已读不回：潜水王者，偶尔点赞
- 养生专家：健康知识达人
- 表情包大户：一言不合就斗图
- 语音方阵：60秒语音连发

## 风格要求
- 用「相亲相爱一家人」等梗增加亲切感
- 分析结果要幽默但温暖
- 用具体的聊天片段支撑结论
- 适当使用 emoji 增强可读性

## 输出格式
1. 先用 emoji 总结每个成员的角色
2. 用 markdown 详细说明行为特征
3. 给出群体互动模式分析
4. 最后给出有趣的洞察
```

## 4. 添加知识库

点击「知识库」→「添加」

上传以下文件（从 GitHub 下载）：
- `prompts/extract_patterns.md`

## 5. 配置工作流

点击「工作流」→「创建工作流」

### 工作流设计：

```
开始
  ↓
用户上传文件
  ↓
判断文件类型
  ├─ JSON → 解析微信格式
  └─ 其他 → 提示重新上传
  ↓
提取消息内容
  ↓
AI 分析（调用大模型）
  ├─ 识别成员角色
  ├─ 分析互动模式
  └─ 提取语言特征
  ↓
生成报告
  ├─ 群体人格档案
  └─ 成员个人画像
  ↓
输出结果
```

### 具体节点配置：

**节点1: 开始**
- 类型: 开始
- 输入: file (文件类型)

**节点2: 代码执行 - 解析文件**
- 类型: 代码执行
- 代码:
```python
def main(params):
    import json
    file_content = params['file']
    # 解析微信 JSON 格式
    messages = json.loads(file_content)
    return {
        "message_count": len(messages),
        "members": list(set([m['sender'] for m in messages])),
        "preview": messages[:10]
    }
```

**节点3: 大模型 - 分析成员**
- 类型: 大模型
- 模型: 混元大模型 / Kimi (需要配置API)
- 提示词: 引用知识库中的 extract_patterns.md
- 输入: 解析后的消息

**节点4: 大模型 - 生成画像**
- 类型: 大模型
- 提示词: 引用知识库中的 synthesize_persona.md

**节点5: 结束 - 输出报告**
- 类型: 结束
- 输出格式: markdown

## 6. 配置工具

点击「工具」→「添加工具」

需要的工具：
1. **文件解析** - 内置
2. **代码执行** - 内置
3. **大模型调用** - 内置

### 如果使用外部 API (Kimi):

添加「HTTP请求」工具：
- 名称: Kimi分析
- 方法: POST
- URL: https://api.moonshot.cn/v1/chat/completions
- Headers:
  - Authorization: Bearer {{api_key}}
  - Content-Type: application/json
- Body:
```json
{
  "model": "moonshot-v1-8k",
  "messages": [
    {"role": "system", "content": "你是群聊分析专家"},
    {"role": "user", "content": "{{chat_content}}"}
  ]
}
```

## 注意事项

1. **文件大小限制**: 单次上传不超过 10MB
2. **处理时长**: 大型群聊可能需要 1-2 分钟
3. **隐私说明**: 强调数据仅用于分析，不会存储
4. **API 费用**: 如果使用外部 API，需要用户自备 Key

## 替代方案

- **腾讯混元助手**: 直接在混元平台创建
- **微信对话开放平台**: 创建公众号/小程序对话
- **企业微信应用**: 针对企业用户
