## 掘金文章

**标题**: 我用 AI 分析了家里「相亲相爱一家人」的800条群聊消息，发现了这些真相

**标签**: AI, Python, 微信, 开源, 人工智能

**内容**:

```markdown
# 用 AI 分析微信群聊，发现每个人的隐藏角色

## 起因

家里有个群叫「相亲相爱一家人」，每天都在发：
- 早上6点的☀️+鸡汤
- "震惊！再不看就删了"的新闻
- 各种养生小贴士

突发奇想：能不能用 AI 分析一下这个群到底什么画风？

## 技术方案

### 1. 数据获取
微信聊天记录导出（使用 WeFlow 等工具），得到 JSON 格式数据。

### 2. 解析处理
```python
# 解析微信导出格式
messages = parse_wechat_json(file)
# 去噪、分段
chunks = create_conversation_chunks(messages)
```

### 3. AI 分析
使用 Kimi API，设计提示词让 AI 识别：
- 成员角色（谁主导话题、谁捧哏、谁潜水）
- 语言特征（口头禅、表情习惯）
- 群体模式（如何表达关心、如何处理分歧）

### 4. 生成报告
输出 markdown 格式的人格档案：
- soul.md（群体人格）
- persona_成员.md（个人画像）

## 分析结果

### 我妈：早安打卡机 ⏰
- 每天早上6点准时发送☀️+鸡汤文案
- 使用祈使句："记得多穿点"、"别忘了吃饭"
- emoji 频率高于文字

### 我爸：转发轰炸机 📰
- 震惊体专业户
- "不转不是中国人"
- 很少原创，但转发量惊人

### 我：已读不回 🙈
- 常年潜水
- 偶尔回个👍维持亲情
- 被@了才回复

### 三姨：养生专家 🏥
- 吃什么致癌、吃什么长寿，了如指掌
- 专业辟谣（同时也传播谣言）
- 关心全家人的健康

## 开源代码

GitHub: https://github.com/zengury/family-analyzer

```bash
pip install family-analyzer
family-analyzer /path/to/群聊.json
```

支持：
- OpenClaw / MCP / CLI
- 微信/WhatsApp 导出
- 自定义 AI 模型

## 应用场景

1. **AI Agent 人格底座**：用真实人格训练 AI
2. **家庭回忆录**：整理聊天精华
3. **社会学研究**：数字民族志数据
4. **纯属娱乐**：看看你家群是什么画风

## 技术亮点

- 4阶段流水线设计
- 支持断点续传
- Batches API 降低成本
- 可扩展的 prompt 系统
